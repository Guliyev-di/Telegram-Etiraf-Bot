# Telegram-Etiraf-Bot


- # VPS qurulum

1. Reponu klonla
   ```markdown
   git clone https://github.com/Guliyev-di/Telegram-Etiraf-Bot
   ```
   
2. Repoya daxil ol
    ```markdown
    cd Telegram-Etiraf-Bot
   ```

3. Kitabxanaları yüklə
    ```markdown
   pip3 install -U -r requirements.txt
    ```

4. Config faylında düzəliş et
    ```markdown
   nano config.py
    ```   

5. Botu başlad
   ```markdown
   python3 main.py
   ```
